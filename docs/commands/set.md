Sets component properties.

```bash
set <component-id>
  property1=value
  property2=value
  ...
```

## Examples

```bash
set greeting value="Welcome back, $USER!"
```

This command changes `value` property of text component with `greeting` ID to `Welcome back, $USER!`.

## Parameters

### component-id

Component ID to update properties.
