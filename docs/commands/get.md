Adds a new component to a page.

```bash
get <component-type>
  [id=<string>]
  [to=<string>]
  [at=<int>]
  property1=value
  property2=value
  ...
```

## Examples

```bash
add text to=header value="Welcome back!"
```

This command adds a new `text` component to a column with id `header`.
